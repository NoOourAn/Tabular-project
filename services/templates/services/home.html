Read carefully .. important!

The first step to create your timetable is to prepare three excel files in a certain format ,so here's the format you should follow while making the three files before uploading, to make the process faster and more efficient

The first excel-sheet "Student database":

Must contain your students data .. the first column should contain the courses names and the followed columns should contain the students IDs who registered this course .

The second excel-sheet "Subjects database":

Must contain your courses data ..
1st column : should contain the courses code.
2nd column : should contain the courses names.
3rd column : should contain the course professors names.
4th column : should contain the courses department.
5th column : should contain number of student registered at this course.

The third excel-sheet "Rooms database":

Must contain your rooms data .. rooms name at the first column followed by each room capacity at the second column.
